##  Project Objective
The objective of this project is to analyze resume keywords and skill demand across different job roles such as Data Scientist, Data Analyst, and Business Analyst. The dashboard helps job seekers understand in-demand skills and assists recruiters in identifying key skill patterns from resumes.

---

##  Dataset Used
- Resume and job keyword dataset containing:
  - Job ID
  - Job Title
  - Company
  - Skill keywords
  - Skill count
- Dataset type: Simulated / anonymized recruitment data  
- Format: CSV / Excel

---

##  Business Questions / KPIs
- Which skills appear most frequently in resumes?
- How do skill requirements vary by job role?
- Which job roles demand the highest number of skills?
- What is the distribution of job roles across companies?
- Which skills are common across multiple job titles?

---

##  Process / Methodology
1. Imported dataset into Power BI
2. Cleaned and transformed data using Power Query
3. Created DAX measures for:
   - Skill count
   - Job count
4. Analyzed keyword frequency across job roles
5. Built role-wise skill distribution visuals
6. Designed interactive slicers for company and job role filtering

---

##  Dashboard Overview
The dashboard includes:
- KPI cards for total skills and total jobs
- Skill count by job title
- Keyword frequency analysis
- Skill distribution by job role
- Job role distribution overview
- Company-wise filtering using slicers

##  Project Highlights
- Power BI, Python, SQL, and Tableau are the most in-demand skills
- Data Scientist roles require the highest skill diversity
- Communication and Excel skills are common across all job roles
- Dashboard provides clear insights for resume optimization
- Helps align resumes with job market requirements

---

##  Final Conclusion
This analysis highlights the importance of aligning resumes with market-demanded skills. By understanding keyword frequency and role-based skill requirements, job seekers can improve resume visibility while recruiters can streamline candidate screening.
